# tkk-case 法律案件全流程处理技能

## 功能介绍

全流程法律案件分析技能，支持：
- 任意方式启动（口述/粘贴/文件）
- 沟通厘清案件
- 调用 tkk-legal-ingest 分析问题（如无答案则用 web-access 检索）
- 生成结构化法律分析文章
- 存入 vault

## 核心框架

### 四类案件对应框架

| 案件类型 | 分析框架 | 核心方法 |
|---------|---------|---------|
| 单一刑事案件 | **两阶层四步法** | 客观违法→客观阻却→主观责任→主观阻却 |
| 单一民商事案件 | **请求权基础分析** | 双视角对抗性分析（原告攻击+被告防御） |
| 单一行政案件 | **逆向三步法** | 锁定结果→追溯依据→对照证据 |
| 刑民交叉案件 | **双轨并行分析** | 刑事轨道+民事轨道同步/协调 |

### 请求权基础分析（民商事）- 双视角对抗性框架

```
【原告视角 - 攻击】              【被告视角 - 防御】
请求权技术分析                  要件审判九步法
├→ 固定原告请求                ├→ 识别原告请求权基础
├→ 系统检索请求权（六类顺位）    ├→ 分解法律规范要件
├→ 规范识别与选择              ├→ 逆向检验各要件
└→ 要件归入确认                └→ 检索可提出的抗辩
```

## 安装方式

```bash
# 克隆仓库
git clone -b tkk-case https://github.com/bosh478/lawyer-skills.git

# 复制技能到 Claude Code 技能目录
cp -r tkk-case ~/.claude/skills/

# 重启 Claude Code 或使用 /skills 重新加载
```

## 使用方法

```
/tkk-case --case "案件素材文件路径" --author 汤康康
```

## 依赖技能

- `tkk-legal-ingest` — 法律文档消化 + 问题分析
- `web-access` — 网络检索（vault无答案时必须调用）
