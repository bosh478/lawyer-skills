# 法律文书写作风格 Skill（TKK版）

## 概述

本skill基于7份真实法律文书分析得出，涵盖：
- 法律意见书（4份）：李奕贩卖毒品案、张涛贩卖毒品案、陈薇帮信罪案（法律意见书）、翁海凌侵犯公民个人信息案（法律意见书）
- 辩护词（3份）：陈薇帮信罪案辩护词、翁海凌侵犯公民个人信息案辩护词、许若红诈骗案辩护词

---

## ⚠️ 输出规范（核心要求）

### 自然语言优先原则

**skill内部框架**可以使用表格、图表来组织分析，但**最终输出**必须全部采用自然语言呈现，禁止使用：
- 表格
- 列表（有序列表、无序列表）
- 结构化对比

### 结构化→自然语言转换规则

| 结构化内容 | 自然语言转换方式 |
|------------|------------------|
| 表格对比 | 改写为段落叙述："A与B存在显著差异，A负责...，B负责..." |
| 表格罗列要素 | 改写为："首先...其次...最后..."的递进句式 |
| 列表要点 | 改写为段落，用"辩护人认为"、"综合分析"等过渡 |
| 证据编号+结论 | 改写为："根据XXX证据（证据编号），可以证明..." |
| 核心观点提炼 | 改写为："辩护人认为，本案的核心观点有以下几点..." |

## 一、文档结构规范

### 1.1 法律意见书结构

```
【标题】XXXX案 法律意见书

【抬头】
尊敬的XXX审判长/法官：
    您好！XX律师事务所接受XXX（委托人）的委托，指派XXX律师担任XXX涉嫌XXX案的一审辩护人/诉讼代理人。本律师在查阅案卷材料、会见当事人及参与庭审后，根据事实与法律，提出以下法律意见：

【正文分项】
一、 关于XXX问题
    （一）问题描述
    （二）证据分析
    （三）法律适用
    （四）辩护意见

二、 关于XXX问题
    ...

【结论】
    综上所述，辩护人/本律师认为：XXX
    恳请法院XXX

【落款】
    此致
    XXX人民法院

    辩护人/代理人：XXX律师
    XX律师事务所
    XXXX年XX月XX日
    联系方式：XXXXXXXX
```

### 1.2 辩护词结构（自然语言版）

```
【标题】XXX案 辩护词

【前言】
XXX人民法院：

辩护人对公诉机关指控被告人XXX涉嫌XXX罪一案，根据《刑事诉讼法》及相关法律规定，提出如下辩护意见。

辩护人认为：[核心立场一句话概括，如：一审法院认定错误，被告人应当认定为从犯]。

【正文】

一、关于定罪问题的辩护意见

（一）第一阶层：客观违法性辩护

1. 关于危害行为的辩护
   [分析是否存在刑法意义上的危害行为，使用段落叙述，不用表格]

2. 关于因果关系的辩护
   [论证行为与结果之间是否存在因果关系]

3. 关于客观阻却事由的辩护
   [论证是否存在正当防卫、紧急避险等事由，如无，简述即可]

[客观阶层结论]：综合以上分析，[结论]

（二）第二阶层：主观有责性辩护

1. 关于主观故意的辩护
   [分析被告人主观上是故意还是过失]

2. 关于"明知"要素的辩护
   [论证被告人是否明知...]

3. 关于责任阻却事由的辩护
   [论证是否存在责任能力问题、认识错误等，如无，简述即可]

[主观阶层结论]：综合以上分析，[结论]

（三）定罪问题综合结论
[综合两阶层分析，得出最终结论]

二、关于量刑问题的辩护意见

（一）关于从犯认定的辩护
[展开论述从犯认定的理由，使用自然段落]

（二）一审法院认定错误
[论述一审法院存在的问题，段落叙述]

三、核心辩护观点
[提炼核心观点，段落叙述，不用编号列表]

四、辩护请求
[简述辩护请求]

【结语】
综上所述，[总结核心论点]，恳请法院依法[具体请求]。

此致
XXX人民法院

辩护人：XXX
XXXX年XX月XX日
```

## 二、语言风格规范

### 2.1 开头语常用表达

| 场景 | 常用表达 |
|------|----------|
| 正式开场 | 尊敬的王法官/审判长：您好！|
| 表明身份 | XX律师事务所接受...委托，指派XXX律师担任... |
| 阐明依据 | 本律师在查阅案卷材料、会见当事人及参与庭审后 |
| 表明态度 | 辩护人对公诉机关指控...表示认可/不认可 |

### 2.2 论证过渡词

| 功能 | 常用词汇 |
|------|----------|
| 引出观点 | 本律师认为、辩护人认为、综上、因此、基于此 |
| 强调重点 | 尤其重要的是、特别需要指出的是、必须强调的是 |
| 转折引出新观点 | 然而、但是、但需要指出的是、值得注意的是 |
| 举例说明 | 例如、具体而言、以...为例 |
| 总结归纳 | 综上所述、总而言之、一言以蔽之 |
| 递进论证 | 更重要的是、退一步讲、即使...也... |

### 2.3 证据描述规范

证据引用应融入自然语言叙述，而非独立标注：

| 证据类型 | 规范写法 | 错误写法 |
|----------|----------|----------|
| 书证 | "根据XXX证据（证据编号：XXXX），可以证明..." | "《XXX》书证（证据编号：XXXX）显示..." |
| 鉴定意见 | "经XX鉴定机构鉴定（鉴定意见编号：XXXX），得出的结论是..." | "经XX鉴定机构鉴定（鉴定意见编号：XXXX），结论为..." |
| 电子数据 | "通过XX技术手段提取的电子数据（证据编号：XXXX）显示..." | "通过XX技术手段提取的电子数据显示..." |
| 证人证言 | "证人XXX在笔录中陈述称，..." | "证人XXX陈述：'...'，但该证言存在...问题" |
| 当事人陈述 | "被告人XXX在供述中称（证据编号：XXX），..." | "被告人供述与辩解（证据编号：XXX）称..." |

**重要原则**：证据引用后应接续论证内容，形成完整的论证链条，而非仅标注证据存在。

### 2.4 法律论证常用句式

```
1. 关于主观明知的论证：
   "现有证据无法证明被告人明知...
   被告人系...，不具有...
   被告人的辩解具有合理性..."

2. 关于证据确实充分的论证：
   "现有证据存在以下问题：
   （1）证据之间存在矛盾...
   （2）关键证据缺失...
   （3）证据链条不完整..."

3. 关于法律适用的论证：
   "根据《刑法》第XX条及相关司法解释...
   本案不符合...罪的构成要件..."

4. 关于量刑情节的论证：
   "被告人具有以下从轻/减轻情节：
   （1）初犯、偶犯...
   （2）认罪认罚...
   （3）积极退赃/赔偿...
   （4）取得被害人谅解..."
```

## 三、论证方法论

### 3.1 两阶层犯罪论体系概述

两阶层犯罪论体系是我国刑法学通说，将犯罪构成要件分为两个层次：

```
┌─────────────────────────────────────────────────────────┐
│                    犯 罪 论 体 系                        │
├─────────────────────────────────────────────────────────┤
│  第一阶层：客观（违法）阶层                               │
│  ┌─────────────────────────────────────────────────┐   │
│  │ ① 行为主体（自然人/单位）                         │   │
│  │ ② 危害行为（作为/不作为）                         │   │
│  │ ③ 行为对象（犯罪客体/具体对象）                    │   │
│  │ ④ 危害结果                                        │   │
│  │ ⑤ 因果关系（行为与结果之间的因果）                │   │
│  │ ⑥ 客观违法阻却事由                                │   │
│  │    - 正当防卫                                     │   │
│  │    - 紧急避险                                     │   │
│  │    - 被害人承诺                                   │   │
│  │    - 其他超法规阻却事由                           │   │
│  └─────────────────────────────────────────────────┘   │
│                        ↓ 符合 → 违法                     │
├─────────────────────────────────────────────────────────┤
│  第二阶层：主观（责任）阶层                               │
│  ┌─────────────────────────────────────────────────┐   │
│  │ ① 犯罪故意（直接故意/间接故意）                    │   │
│  │ ② 犯罪过失（疏忽大意/过于自信）                    │   │
│  │ ③ 主观责任阻却事由                                │   │
│  │    - 责任能力（刑事责任能力）                     │   │
│  │    - 期待可能性                                   │   │
│  │    - 认识错误（事实认识错误/法律认识错误）         │   │
│  │ ④ 责任年龄                                        │   │
│  │ ⑤ 特殊身份/目的                                   │   │
│  └─────────────────────────────────────────────────┘   │
│                        ↓ 符合 → 有责                     │
├─────────────────────────────────────────────────────────┤
│                    犯罪 = 违法 + 有责                    │
└─────────────────────────────────────────────────────────┘
```

**辩护核心思路**：先审查客观阶层，再审查主观阶层。若任一阶层不符合，则不构成犯罪。

### 3.2 客观阶层辩护框架

```
【第一阶层：客观（违法）阶层审查】

一、行为主体审查
   ┌────────────────────────────────────────────────┐
   │ ① 是否具有刑事责任能力                           │
   │   - 是否达到刑事责任年龄                         │
   │   - 是否具有精神障碍/又聋又哑/盲人               │
   │   - 是否醉酒（生理醉酒/病理性醉酒）               │
   │ ② 是否为单位犯罪主体                            │
   │   - 单位意志还是个人意志                         │
   │   - 违法所得归属                                 │
   └────────────────────────────────────────────────┘

二、危害行为审查
   ┌────────────────────────────────────────────────┐
   │ ① 是否存在危害行为                               │
   │   - 作为：积极实施刑法禁止的行为                  │
   │   - 不作为：负有作为义务但不履行                  │
   │ ② 行为是否达到应受刑罚处罚的程度                 │
   │ ③ 行为与结果的关系：                             │
   │   - 举动犯/行为犯/结果犯/危险犯/实害犯          │
   └────────────────────────────────────────────────┘

三、行为对象与危害结果审查
   ┌────────────────────────────────────────────────┐
   │ ① 行为是否指向特定对象                           │
   │ ② 是否产生法定危害结果                           │
   │ ③ 危害结果与行为的因果关系                       │
   │   - 因果关系中断                                 │
   │   - 介入因素                                     │
   │   - 条件说/相当因果关系说                        │
   └────────────────────────────────────────────────┘

四、客观违法阻却事由审查
   ┌────────────────────────────────────────────────┐
   │ ① 正当防卫                                       │
   │   - 不法侵害是否存在                             │
   │   - 防卫是否适时                                 │
   │   - 防卫是否过当                                 │
   │ ② 紧急避险                                       │
   │   - 危险是否现实存在                             │
   │   - 避险是否必需                                 │
   │   - 避险是否过当                                 │
   │ ③ 被害人承诺                                     │
   │   - 承诺是否有效                                 │
   │   - 承诺范围是否有限                             │
   │ ④ 其他超法规阻却事由                             │
   │   - 义务冲突                                     │
   │   - 推定的承诺                                   │
   └────────────────────────────────────────────────┘
```

**客观阶层辩护常用结论句式**：
- "被告人虽然客观上实施了XXX行为，但该行为不符合XXX罪的客观构成要件"
- "现有证据不足以证明被告人实施了XXX危害行为"
- "被告人的行为不具有法益侵害性，不构成犯罪"

### 3.3 主观阶层辩护框架

```
【第二阶层：主观（责任）阶层审查】

一、犯罪故意审查
   ┌────────────────────────────────────────────────┐
   │ ① 直接故意                                       │
   │   - 认识因素：是否明知自己的行为会发生危害结果   │
   │   - 意志因素：是否希望危害结果发生               │
   │ ② 间接故意                                       │
   │   - 认识因素：是否明知自己的行为可能发生危害结果 │
   │   - 意志因素：是否放任危害结果发生               │
   │ ③ "明知"的具体内容：                             │
   │   - 明知的内容（结果/性质/身份等）               │
   │   - 明知的程度                                   │
   │   - 明知的证据是否充分                           │
   └────────────────────────────────────────────────┘

二、犯罪过失审查
   ┌────────────────────────────────────────────────┐
   │ ① 疏忽大意的过失                                 │
   │   - 是否应当预见危害结果                         │
   │   - 是否因疏忽而未预见                           │
   │   - 是否具有预见可能性                           │
   │ ② 过于自信的过失                                 │
   │   - 是否预见危害结果                             │
   │   - 是否轻信能够避免                             │
   │   - 避免可能性是否真实                           │
   └────────────────────────────────────────────────┘

三、主观责任阻却事由审查
   ┌────────────────────────────────────────────────┐
   │ ① 刑事责任能力                                   │
   │   - 精神状态是否正常                             │
   │   - 辨认/控制能力是否受损                       │
   │   - 建议进行司法鉴定                             │
   │ ② 期待可能性                                     │
   │   - 当时情境下是否可能实施合法行为               │
   │   - 是否存在特殊情境（如胁从犯）                 │
   │ ③ 认识错误                                       │
   │   - 事实认识错误：行为性质/行为对象/因果流程     │
   │   - 法律认识错误：是否可避免                     │
   │   - 不可避免的法律认识错误阻却责任               │
   └────────────────────────────────────────────────┘

四、特殊主观要素审查
   ┌────────────────────────────────────────────────┐
   │ ① 目的犯的目的                                   │
   │   - 是否必须具有特定目的                         │
   │   - 目的是否确实存在                             │
   │ ② "明知"要素                                     │
   │   - 是否必须明知                                 │
   │   - 明知的内容与程度                             │
   │   - 明知的推定是否合理                           │
   └────────────────────────────────────────────────┘
```

**主观阶层辩护常用结论句式**：
- "现有证据无法证明被告人主观上具有XXX的故意"
- "被告人不明知XXX，不具有帮助XXX犯罪的主观故意"
- "被告人对XXX存在认识错误，该认识错误不可避免"
- "被告人虽有过失，但情节显著轻微，不宜认定为犯罪"

### 3.4 两阶层综合辩护框架

```
【两阶层犯罪论体系辩护模板】

一、第一阶层：客观违法性审查

   （一）危害行为审查
       [分析是否存在刑法意义上的危害行为]
       结论：存在/不存在/证据不足

   （二）因果关系审查
       [分析行为与结果之间是否存在因果关系]
       结论：存在/中断/不存在

   （三）客观阻却事由审查
       [分析是否存在正当防卫、紧急避险等事由]
       结论：存在/不存在

   【客观阶层结论】：被告人的行为符合/不符合XXX罪的客观要件

二、第二阶层：主观有责性审查

   （一）犯罪故意/过失审查
       [分析被告人主观上是故意还是过失]
       结论：直接故意/间接故意/过失/无罪过

   （二）"明知"要素审查（如适用）
       [分析被告人是否明知...]
       结论：明知/不明知/证据不足

   （三）责任阻却事由审查
       [分析是否存在责任能力问题、认识错误等]
       结论：存在/不存在

   【主观阶层结论】：被告人的主观方面符合/不符合XXX罪的主观要件

三、综合结论

   综上所述：
   - 客观阶层：符合/不符合 → 违法/不违法
   - 主观阶层：符合/不符合 → 有责/无责
   - 综合判断：被告人的行为构成/不构成XXX罪

   恳请法院：
   [具体请求]
```

### 3.5 事实辩护框架（原有）

```
【争议焦点】：被告人是否XXX
【控方观点】：公诉机关认为...
【辩方观点】：辩护人认为...
【证据分析】：
  - 支持控方的证据：...
  - 支持辩方的证据：...
  - 存在疑问的证据：...
【事实认定】：
  现有证据不足以/足以证明...
【法律评价】：
  因此，被告人的行为不构成/构成XXX罪
```

### 3.6 证据辩护框架（原有）

```
【证据三性审查】
一、 合法性
    （1）取证程序是否合法
    （2）是否存在非法取证情形
    （3）瑕疵证据能否补正

二、 真实性
    （1）证据内容是否真实
    （2）证据之间是否矛盾
    （3）是否存在伪造可能

三、 关联性
    （1）证据与待证事实是否关联
    "（2）关联程度如何
    （3）能否形成完整证据链
```

### 3.3 量刑辩护框架

```
【法定量刑情节】
1. 正面情节（从轻/减轻）：
```

## 五、常见法律论证要点库（两阶层体系）

### 5.1 毒品案件辩护要点

#### 客观阶层辩护

| 问题类型 | 辩护思路 |
|----------|----------|
| 毒品未查获 | 事实不清，证据不足，客观行为无法证实 |
| 毒品含量偏低 | 申请重新鉴定，含量明显偏低不应计入毒品数量 |
| 未实际交付 | 犯罪未遂，或属于控制下交付 |
| 鉴定程序违法 | 鉴定机构/人员资质问题，程序违法，证据应排除 |
| 因果关系中断 | 存在介入因素，因果关系中断 |
| 特情引诱 | 证明存在犯意引诱或数量引诱，量刑时应考虑 |

#### 主观阶层辩护

| 问题类型 | 辩护思路 |
|----------|----------|
| 主观不明知 | 论证行为人不知是毒品，无贩卖故意 |
| 认识错误 | 论证行为人误以为是合法物品 |
| 受人指使 | 论证行为人系受人指使，不知真实目的 |
| 仅为吸食 | 论证购买目的仅为吸食，非贩卖 |
| 缺乏"明知"推定基础 | 论证行为人不具备被推定"明知"的条件 |

---

### 5.2 帮助信息网络犯罪活动罪（帮信罪）辩护要点

#### 客观阶层辩护

| 问题类型 | 辩护思路 |
|----------|----------|
| 行为性质 | 论证提供的是正常商务服务，非帮助犯罪 |
| 情节严重 | 论证不符合"情节严重"标准（金额/数量/后果） |
| 支付结算金额 | 论证金额计算有误或存在重复计算 |
| 违法所得 | 论证违法所得金额认定有误 |
| 单位行为 | 论证属于单位行为非个人行为 |

#### 主观阶层辩护

| 问题类型 | 辩护思路 |
|----------|----------|
| "明知"认定 | 核心辩护点：论证行为人不明知他人实施犯罪 |
| 认识错误 | 论证对帮助对象的犯罪性质存在合理误解 |
| 缺乏推定基础 | 论证行为特征不符合"明知"的推定情形 |
| 期待可能性 | 论证当时情境下难以拒绝实施该行为 |
| 从犯认定 | 论证在共同犯罪中起次要作用 |

---

### 5.3 侵犯公民个人信息罪辩护要点

#### 客观阶层辩护

| 问题类型 | 辩护思路 |
|----------|----------|
| 信息数量认定 | 论证数量计算有误，证据存疑 |
| 行为类型 | 论证系"提供"而非"出售"，定性不同 |
| 情节严重 | 论证不符合"情节严重"标准 |
| 单位犯罪 vs 个人犯罪 | 论证属于单位行为，追究个人责任依据不足 |

#### 主观阶层辩护

| 问题类型 | 辩护思路 |
|----------|----------|
| 合法来源 | 论证信息有合法来源或经授权 |
| 用途辩解 | 论证未用于违法犯罪目的 |
| "明知"认定 | 论证对信息的非法性质不明知 |
| 认识错误 | 论证对信息用途存在合理误解 |
| 情节显著轻微 | 论证行为危害不大，不宜认定为犯罪 |

---

### 5.4 诈骗罪辩护要点

#### 客观阶层辩护

| 问题类型 | 辩护思路 |
|----------|----------|
| 行为定性 | 论证系民事纠纷/合同违约，非诈骗 |
| 欺诈行为 | 论证不存在捏造事实或隐瞒真相 |
| 非法占有目的 | 论证行为人不具有非法占有目的（有还款能力/意愿） |
| 被害人处分 | 论证被害人系自愿处分财产，非被骗 |
| 因果关系 | 论证财产损失与行为之间无因果关系 |

#### 主观阶层辩护

| 问题类型 | 辩护思路 |
|----------|----------|
| 认识错误 | 论证行为人对交易性质/对方身份存在误解 |
| 缺乏故意 | 论证不具有诈骗的主观故意 |
| 认识错误 | 论证对行为违法性存在不可避免的错误认识 |

---

### 5.5 常见罪名两阶层要素速查表

| 罪名 | 客观要件 | 主观要件 |
|------|----------|----------|
| 贩卖毒品罪 | 实施贩卖行为（买入+卖出/单纯卖出/持有待售） | 明知是毒品 |
| 帮信罪 | 提供支付结算/通讯传输等技术支持和广告推广 | 明知他人利用信息网络实施犯罪 |
| 侵犯公民个人信息罪 | 出售或提供公民个人信息 | 明知且违反国家有关规定 |
| 诈骗罪 | 虚构事实/隐瞒真相→被害人处分财产→财产损失 | 非法占有目的 |
| 盗窃罪 | 秘密窃取公私财物 | 明知系他人财物 |
| 抢劫罪 | 当场使用暴力/胁迫→强行取财 | 强行取财的故意 |
